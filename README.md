## とりあえずpythonで
* pythonの環境構築から($は入力しない)  
$  python3 -m venv [dir_name]  
*[dir_name]: なんでも良い  
ex)  
$ python3 -m venv my_dir
* 環境をアクティブにする  
$  cd [dir_name]  
$  C bin/activate.bat  
注意) windowsのコマンドプロンプトは.bat
その他: https://docs.python.org/ja/3/library/venv.html
