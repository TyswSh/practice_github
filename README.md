## とりあえずpythonで
1. pythonの環境構築から($は入力しない)  
$  python3 -m venv [dir_name]  
*[dir_name]: なんでも良い  
ex) $ python3 -m venv my_dir  
2. 環境をアクティブにする  
$  cd [dir_name]  
$  source /bin/activate  
*上記のはbash/zshなのでwindowsの人は気を付ける
$ mkdir src && cd src
